About Stream Tester

Stream Tester is a tool designed for developers and streaming enthusiasts to debug HLS and MPEG-DASH video streams. Built with the power of Shaka Player, it provides a consistent environment to validate manifests, test DRM configurations, and ensure playback stability.

Features include custom playlist (M3U) parsing, direct URI testing, and support for Widevine and ClearKey DRM systems.
